## 锁定风格

- 西方奇幻领地地图，俯视/近正交视角，写实细腻的手绘游戏地形质感。
- 晚秋、低饱和的橄榄褐、赭褐和暗金色为主；整体偏冷暗，不做鲜艳橙红“旅游明信片”效果。
- 地貌骨架必须继承 L0：左上暗色针叶林，右上灰色岩脊，右侧至下缘蜿蜒溪流，中部为可供后续建筑叠加的开阔缓坡/草土地带。
- L1/L2 是对同一地理位置的真实细化，不是重新构图；放大时地貌、溪流、林缘、岩脊必须落在对应位置。
- L2 采用 `autumn-l2-production-tile-v2.png` 的微观密度：细碎土壤、枯草、砾石和少量低矮灌木，避免重复纹样、刷痕和均匀噪点。

## 禁用项

- 禁止建筑、房屋、城堡、农田、营地、桥梁、道路、围栏、路牌、灯火、车辆、人物和其他人工物。
- 禁止文字、坐标、水印、UI、边框、网格线和发光标记。
- 禁止积雪、冰层、冬季白霜或明显冬季色调。
- 禁止把尺度预览里的茅草屋烘焙进地图；建筑应由前端作为独立透明图层叠加。
- 禁止仅对 L0 做普通放大、锐化或 AI 超分后充当 L1/L2；必须生成真实新增细节。
- 禁止在未确定坐标时将原型文件登记为完成瓦片。
